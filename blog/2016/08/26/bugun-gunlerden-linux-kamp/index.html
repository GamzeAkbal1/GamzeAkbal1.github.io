Önce Bağımlılık Yönetimi Nedir ? Ona bakalım sonra da kampta ödev olarak verilen ruby oyunumuza;

Bağımlılık Yönetimi “Dependency management” x geminin çalışması için a,b,c,d,e gemine ihtiyaç var ise buna Bağımlılık Yönetimi (DM) denir. Bunları tek tek gem install a, gem install b yazmak yerine “ bundle ” şekilde indirmek en mantıklısıdır bir nevi paket şeklinde.Zaten bundle bohça, paket anlamına gelmektedir. Bundle init yazdığımızda gem file dosyasını oluşturur ondan sonra dosyamıza girerek (Linux komutlarından hatırlarsınız cd) bundle install yazarak içindeki gemleri hepsini kurabiliriz

Burada iki amaç var kısaca;

  • 1)Gemleri kısaca kurmak
  • 2)Gemlerin versiyonları çok hızlı geliştiği için
